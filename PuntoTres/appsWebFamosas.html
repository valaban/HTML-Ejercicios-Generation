1. Facebook

Fecha de lanzamiento: 4 de febrero de 2004

Empresa: Meta (anteriormente Facebook, Inc.)

Descripción: Facebook es una plataforma de redes sociales que permite a los usuarios conectarse con amigos y familiares, compartir contenido, y descubrir nuevos intereses.

Tecnologías implementadas:

  • React
  • GraphQL
  • PHP
  • MySQL

2. YouTube

Fecha de lanzamiento: 14 de febrero de 2005

Empresa: Google

Descripción: YouTube es una plataforma de videos en línea que permite a los usuarios subir, ver y compartir videos.

Tecnologías implementadas:

  • Python
  • HTML5
  • JavaScript
  • C++

3. Instagram

Fecha de lanzamiento: 6 de octubre de 2010

Empresa: Meta

Descripción: Instagram es una red social centrada en compartir fotos y videos, con un enfoque en la creatividad y la expresión visual.

Tecnologías implementadas:

  • Django
  • Python
  • React Native
  • PostgreSQL

4. WhatsApp

Fecha de lanzamiento: 3 de mayo de 2009

Empresa: Meta

Descripción: WhatsApp es una aplicación de mensajería instantánea que permite a los usuarios enviar mensajes de texto, voz, imágenes y videos de forma gratuita.

Tecnologías implementadas:

  • Erlang
  • XMPP
  • JavaScript
  • SQLite

5. Twitter

Fecha de lanzamiento: 21 de marzo de 2006

Empresa: X Corp. (anteriormente Twitter, Inc.)

Descripción: Twitter es una red social y servicio de microblogging en el que los usuarios publican y leen mensajes cortos llamados 'tweets'.

Tecnologías implementadas:

  • Ruby on Rails
  • JavaScript
  • Scala
  • MySQL
